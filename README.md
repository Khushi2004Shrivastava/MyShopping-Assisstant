# 🛒 MyShoppingAssistant

**MyShoppingAssistant** is a smart retail solution that brings the power of digital shopping to physical stores. It simplifies the in-store shopping experience by combining QR technology with intelligent tracking and user-friendly features.

## 🚀 Features

- **📱 QR-Based Product Scanning**  
  Shoppers scan QR codes on products to:
  - Add items to a virtual cart
  - Check product prices
  - Get a real-time bill estimate

- **🛍️ Buy Now or Save for Later**  
  Users can either:
  - Complete the purchase instantly via online checkout
  - Save items to buy later from home

- **🔔 Expiry & Restock Alerts**  
  QR codes are linked to the user’s mobile app to:
  - Notify about upcoming product expiries
  - Set reminders for restocking essentials

---

## 🧰 Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Flask (Python)  
- **Database:** JSON-based product store (can be replaced with SQL/MongoDB)  
- **Other Tools:** Bootstrap, Jinja2 templates
